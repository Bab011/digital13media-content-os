# Workflows & Processes

This folder documents the operational workflows and routines used to run the Digital13Media content engine.

## Core Workflows
*   `insight-extraction-workflow.md` - How to turn a raw voice note, email, or GDS discovery into a structured knowledge asset.
*   `ai-generation-workflow.md` - Step-by-step instructions on how to prompt Google AI Studio using our knowledge base files as source context.
*   `brand-voice-review-checklist.md` - The final quality control checklist to run every piece of content through before it is scheduled.
