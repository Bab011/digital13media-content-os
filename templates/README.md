# Content Templates

This folder holds the structured blueprints and visual/text templates for Digital13Media.

## Structure
*   `linkedin-formats/` - Post blueprints (e.g., the "Rhythm Pattern" layout, listicle structures, and authority hooks).
*   `newsletter-outlines/` - Clean formats for structuring email newsletter issues.
*   `case-studies/` - Structured frameworks for transforming a successful client project into a case study.
